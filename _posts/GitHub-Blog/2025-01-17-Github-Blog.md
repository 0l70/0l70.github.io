# GitHub로 블로그 포스팅하기
## 1. 맘에 드는 테마를 찾아온다. 
깃허브로 연 후 상단 3번째에 Fork를 누른다. 사용자_이름.github.io로 설정한 후 _config.yml의 주소도 바꿔준다.

## 2. 포스팅 준비
add file을 한 후 이름을 _posts/year-mm-dd-title.md로 설정한다.

## 3. 포스팅 
Commit changes를 눌러 포스팅한다.
